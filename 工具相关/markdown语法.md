注意：快捷键根据工具不同而不同，本处所列为本工具的快捷键设置。 

- ==加粗==：
	`**内容**`  或者 `__内容__`   **显示效果**    __显示效果__  `ctrl + B`

- ==倾斜==
	`*内容*`  或者 `_内容_`    *显示效果*     _显示效果_   `ctrl + I`

- ==高亮==
	`==内容==`   ==显示效果==         `ctrl + H`

- ==删除线==
	`~~内容~~`   ~~显示效果~~         `ctrl + D`

- ==下划线==
	`<u>内容<u>`  

- ==代码==
	\` 内容\`     `显示效果`           `ctrl + ~` 

- ==插入代码块==
	前后用 \`\`\`            `ctrl + shift + ~` 
```
    显示效果
```

- ==代办==
	- [ ] 内容  =  `- + 空格 + [ ] + 空格 + 内容` 
	- [x] 内容 = `- + 空格 + [x] + 空格`

- ==放大文本==
	`<big>内容</big>`   <big>显示效果</big>

- ==缩小文本==
	`<small>内容</small>`  <small>显示效果</small>

- ==文本颜色==
	`<font color=orange>内容</font>`   <font color=orange>这是一段橘色文本</font>
		 color 里的颜色支持 英文单词，**[16进制](http://c.runoob.com/front-end/55 "可跳转至菜鸟教程了解")，[rgb](https://www.runoob.com/cssref/func-rgb-css.html "可跳转至菜鸟教程了解")，[rgba](https://www.runoob.com/cssref/func-rgba.html "可跳转至菜鸟教程了解")**
		 

- ==Emoji表情==
	😀  🤐 😫 😴 🤩......     `Win + .`

- ==图片缩放==：
1. 标准写法：
```
<img src="./image/xiaoyuehan.png" width="20%" height="auto">
```
<img src="./image/xiaoyuehan.png" width="20%" height="auto">

2. 非标准写法：
```
<img src="./image/xiaoyuehan.png" style="zoom: 20%;" />
```
<img src="./image/xiaoyuehan.png" style="zoom: 20%;" />

3. 本工具的写法
```
![[./image/xiaoyuehan.png|200]]
```
![[./image/xiaoyuehan.png|200]]
